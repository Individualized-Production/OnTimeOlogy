# The OnTime Ontology

The OnTime Ontology provides a semantic framework for integrating external open mobility data with Just-in-Time (JIT) construction logistics concepts.

The ontology enables interoperability between heterogeneous mobility datasets and lean construction processes through reusable semantic concepts and linked data principles.
